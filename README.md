# Toku App

A simple Flutter application to learn basic Japanese vocabulary with pronunciation.


## Features

- Learn Japanese words for **Numbers, Family Members, Colors, and Common Phrases**
- Displays English and Japanese text for each word or phrase
- Audio pronunciation for each item
- Clean and simple UI with reusable components
- Navigation between different categories
